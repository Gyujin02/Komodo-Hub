# Komodo Hub

Komodo Hub is a  prototype  for the final team project submission.  

## Main Features

- Role-based login and dashboards
- School registration and approval flow
- Teacher activity creation and student submission flow
- Community sighting report feature
- Public pages for schools, communities, and species
- Foundation admin dashboard for review and management

## Database Note

The final prototype uses SQLite for easier local setup, testing, and demonstration rather than mySQL.

## Requirements

- Python 3
